# Marp Player

Free online Markdown-to-Slides player powered by [Marp Core](https://marp.app/).

🌐 **Live:** [marp-play.techbuzzz.me](https://marp-play.techbuzzz.me)

## Features
- 🎯 Full Marp syntax, ✨ Slide animations, 📄 PDF export, 📺 Fullscreen
- 🖱️ Click-to-advance, 🔄 Auto-play, ⌨️ Keyboard navigation, 📝 Speaker notes
- 📱 Mobile-first, 🔍 SEO + LLM indexing, 🐳 Docker

## Slide Animations
```markdown
<!-- _animateIn: bounceInLeft -->
<!-- _animateOut: fadeOut -->
```
Presets: None, Fade, Bounce, Slide, Flip, Per-slide

## Quick Start
```bash
cd nextjs_space && yarn install && yarn dev
```
